# Buscador de peliculas

Esta aplicación fue inspirada en una prueba tecnica desarrollada en react con los siguientes requerimentos
- Mostrar un input para buscar la pelicula y un boton para buscar
- listado de las peliculas
- mostrar todo en una grid responsiva

PRIMERA ITERACIÓN

- evitar que la busqueda se haga dos veces seguidas
- hacer la busqueda al escribir en el input
- evitar que se haga la busqueda al escribir(debaunce)
